## sharpen

Add sharpness to an image. If not specified, the default value is `5`.

> The `sharpen` parameter can be added to the querystring as either `sh` or `sharpen`

`https://cdn.example.com/images/beach.jpg?sharpen=25`

**Default amount = 5**
